# Portfolio

Config-driven Next.js portfolio with a 3D cylinder carousel UI.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project structure

- `config/` — layout, theme, animations, projects, and scene settings
- `components/` — shell, cylinder carousel, scene, and navigation
- `lib/` — config resolvers, hooks, and layout math
